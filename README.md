# ThreadStyle - Cloth Website

A modern, responsive e-commerce website for clothing and fashion items.

## Features

✨ **Modern Design**
- Clean, professional interface
- Responsive layout (mobile, tablet, desktop)
- Smooth animations and transitions

🛍️ **Shopping Features**
- Browse featured clothing collection
- Add items to cart
- View cart with total calculation
- Local storage to persist cart

📱 **User-Friendly**
- Sticky navigation header
- Smooth scrolling between sections
- Contact form
- Social media links

🎨 **Design Highlights**
- Modern gradient hero section
- Hover effects on products
- Modal shopping cart
- Smooth animations

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive functionality
- `README.md` - This file

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Browse the products and add items to cart
4. Use the contact form to get in touch
5. Navigate using the sticky header menu

## Sections

- **Home** - Hero section with call-to-action
- **Shop** - Featured product collection
- **About** - Information about ThreadStyle
- **Contact** - Contact form

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- JavaScript (Vanilla, no frameworks)
- LocalStorage API

## Customization

You can easily customize:
- Product items in `script.js`
- Colors in `styles.css`
- Content in `index.html`
- Add more features as needed
